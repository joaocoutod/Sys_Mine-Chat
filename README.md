# Informações importantes

O arquivo para criação do banco e tabelas: `Sys_Mine-Chat/DB/chat.sql`

<hr> 

O banco de dados pode ser configurado em: `Sys_Mine-Chat/app/config.php`

```php
$db_host = "localhost";
$db_db = "chat";
$db_user = "root";
$db_pass = "";
```
